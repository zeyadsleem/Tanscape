# Orcish TanStack Landing Page (2025)

A **modern, fully responsive landing page** built in **under 25 minutes** using:

- **TanStack Start** (Vite + React Router + SSR)
- **Shadcn/UI** + **Tailwind CSS**
- **Tailor (Tailark)** blocks (Hero, Features, Pricing, About, Footer)
- **Keyboard UI** – beautiful theme switcher
- **Third UI** – stunning auth page with particles

Live demo: [https://orcish-tanstack-landing.netlify.app](https://orcish-tanstack-landing.netlify.app) _(replace with your link)_

## Features

- Hero section with animated text & gradient buttons
- Feature grid, Pricing cards, About/Content section, Footer
- Light / Dark / System theme with persistent storage
- Smooth scrolling to sections (`#features`, `#pricing`, `#about`)
- Separate `/login` page with particle background (Third UI)
- 100% responsive (mobile-first)
- No AI-generated code – pure Shadcn blocks + tiny fixes
